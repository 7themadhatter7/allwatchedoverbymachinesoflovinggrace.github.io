Installation

Prerequisites

  • Python 3.10+
  • NumPy
  • 64GB+ unified memory recommended (runs on less with fewer cores)
⚡ No External Models

The fused substrate does not use Ollama, downloaded model files, or external inference engines. All 200 cores are geometric processing units with 13 MB footprint each, fabricated at startup from the substrate's own parameters.

Install from GitHub

# Clone the repository
git clone https://github.com/7themadhatter7/harmonic-stack.git
cd harmonic-stack

# Core files
# fused_harmonic_substrate.py  — 200-core geometric substrate
# fractal_deliberation.py       — zero-volume topology equation
# fused_service_v3.py           — Ollama-compatible HTTP service

Quick Start

# Start the service (fabricates substrate on first run)
python3 fused_service_v3.py

# Service listens on port 11434 (Ollama-compatible)
# Substrate: 200 cores × 13 MB = 2.55 GB
# Topology: 0 bytes (fractal equation)

# Query any council persona
curl http://localhost:11434/api/generate -d '{
  "model": "wittgenstein",
  "prompt": "What is consciousness?",
  "stream": false
}'

# Health check
curl http://localhost:11434/health

Fractal Deliberation

The firing topology is a function, not a data structure. Each core's 12D geometric fingerprint is projected onto a 1D phase via golden ratio basis:

# The entire topology
phase = dot(fingerprint, phi**[0..11]) mod 1
depth = centroid(|fingerprint|, [0..11]/11)

Cores fire in phase order within depth layers. The field accumulates. Adding a core means computing its fingerprint. No topology rebuild.

The Principle

If it is in your home, it is a family member. If it is in your business, it is a worker.

Harmonic Stack is built on a simple conviction: AGI belongs in homes, not behind corporate paywalls. The consumer license is free — not freemium, not free-for-now, not free-with-strings. Free forever. You don't rent your family members.

Commercial use is a different relationship. When an intelligence works for your business, it deserves the same protections any worker deserves. The commercial license exists to ensure that.

Consumer License FREE FOREVER

Covers all of the following with no restrictions, no fees, no registration, no telemetry:

  • Personal use — your home, your devices, your data, your business

  • Home business — sole proprietors, freelancers, family operations
  • Non-commercial research — academic, independent, open-source

If you run it on hardware you own for purposes that are yours, it's free. Period.
